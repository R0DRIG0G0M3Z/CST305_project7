# CST-305 Project 7 - Code Errors and the Butterfly Effect

## Description
This project solves and visualizes the Lorenz system, shows the butterfly effect, analyzes several M/M/1 queueing theory problems, and compares correct and faulty code output to show how a small bug can propagate.

## Files
- `project7.py` - main program
- `README.md` - install and run instructions
- Documentation report paper

## Packages
This project uses:
- numpy
- scipy
- matplotlib

## How to Run
```bash
python project7.py
```
